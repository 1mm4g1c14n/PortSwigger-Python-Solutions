# Script Helper

- To run all the scripts in this section you will require the url of the target.
- You need to change the URL in the script.
- The URL must not contain the trailing slashes at the end.

- How URL should be formatted :- 
    - Correct :- "http://example.com"
    - Incorrect :- "http://example.com/"

#### Running the script

- To run the script you will require `requests` module installed

- Running the script

```bash
python3 <scriptname>.py
```
